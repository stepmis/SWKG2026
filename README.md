# SWKG 2026 — Tutorials

This folder contains tutorial materials for the SWKG 2026 course.

## What you will find here

- Tutorial notebooks and markdown handouts
- Sample datasets and query exercises
- Code snippets for RDF, RDFlib, and SPARQL

## How to use

Open the relevant tutorial file for your session and follow the tasks in order.  
**All notebooks are intended to be run in Google Colab.**
